About NutriGuide

Project Overview

NutriGuide is a Smart Nutrition Analysis and Health Recommendation System developed as a Final Year Project.

The application analyzes nutritional information of foods and provides personalized health recommendations based on user profile, BMI, medical conditions, and fitness goals.

Features

  • Food Nutrition Analysis
  • BMI Calculation
  • Health Score Assessment
  • Food Quality Rating
  • Disease-Based Recommendations
  • Exercise Suggestions
  • Diet Planning
  • PDF Report Generation

Technologies Used

  • Python
  • Flask
  • HTML
  • CSS
  • Pandas
  • ReportLab
